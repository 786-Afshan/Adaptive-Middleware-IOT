# Adaptive-Middleware-IOT
# Adaptive Middleware Optimization for Energy-Efficient Communication in IoT Smart Home Environments

This repository contains the Python simulation code developed for the research paper:

**Adaptive Middleware Optimization for Energy-Efficient Communication in IoT Smart Home Environments**

## Overview

The proposed framework introduces:

* Redundant packet filtering
* Adaptive scheduling
* Priority-aware communication handling
* Energy-efficient middleware optimization

for improving communication efficiency in IoT smart home environments.

## Performance Improvements

The experimental results demonstrate:

* 52.14% reduction in energy consumption
* 41.60% reduction in communication overhead
* 14.01% improvement in communication latency

compared with conventional middleware approaches.

## Implementation

The simulation framework was implemented in Python using:

* NumPy
* Pandas
* Matplotlib
